Q-GAIN: Quantum Gas Analysis and Inference
================================================

Q-GAIN is a Python library serving as the foundation of a growing modular architecture to support a wider range of machine learning and physically informed analysis applications.

It is featured with two built-in detectors: an updated re-implementation of the SolDet package as a subpackage and a basic vortex detector. The SolDet subpackage is suitable for classification, object detection, and physics informed statistical methods on solitons in Bose-Einstein Condensates. The vortex detector is suitable for identifying the positions of vortices in Bose-Einstein Condensates.
